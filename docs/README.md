# docs/ フォルダ - ハンズオン用 PDF ドキュメント

このフォルダには、**野村アセットマネジメントの目論見書 PDF** を格納します。
ノートブック `02b_fund_docs.ipynb` が GitHub の raw URL からこのフォルダの PDF を取得し、Snowflake の内部ステージにアップロードします。

---

## 格納するファイル一覧

| ファイル名（このフォルダに配置） | ファンド名 | 取得元 |
|---|---|---|
| `nomura_index_topix.pdf` | 野村インデックスファンド・TOPIX（交付目論見書） | 野村AM公式サイト |
| `nomura_world_reit.pdf` | 野村世界REITファンド（毎月分配型）（交付目論見書） | 野村AM公式サイト |
| `nomura_esg_fund.pdf` | 野村ESGスコア・エクスポージャー・ファンド（交付目論見書） | 野村AM公式サイト |

> ファイルは 3 本でなくても構いません。1〜5 本程度であればそのまま動作します。
> ファイル名は上記に合わせてリネームしてください。

---

## ダウンロード手順

1. [野村アセットマネジメント 投資信託詳細検索](https://www.nomura-am.co.jp/fund/search/) にアクセス
2. 上記ファンド名で検索
3. 各ファンドの詳細ページから「**投資信託説明書（交付目論見書）**」PDF をダウンロード
4. このフォルダに配置し、上記ファイル名にリネーム
5. `git add docs/*.pdf && git commit -m "Add fund prospectus PDFs" && git push`

## 注意事項

- 本ノートブックで使用する目論見書は**一般公開されている公開情報**です
- ハンズオン（学習目的）での使用に限定してください
- 実際の投資判断の根拠には使用しないでください

---

## ファイルが存在しない場合（デモ用フォールバック）

PDF が用意できない場合は、ノートブック内のデモ用セクションで  
`LOAN_PRODUCT_DOCS` テーブルのデータを代替として使用することも可能です。
